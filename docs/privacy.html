Privacy Policy

AGENTS.md Bridge · Last updated 2026-05-30

AGENTS.md Bridge collects no data. None.

The plugin runs a single local script when a Claude Code session starts. That script:

  • Reads AGENTS.md files in your current project directory.
  • Writes local CLAUDE.md files next to them and updates your local .git/info/exclude.
  • Prints a one-line status message.

That is the entirety of its behavior. Specifically, the plugin does not:

  • Make any network requests.
  • Send telemetry, analytics, or usage data anywhere.
  • Read, collect, or transmit your source code, file contents, credentials, or personal information.
  • Store data outside your own project directory.
  • Use cookies or any tracking technology.

All processing happens locally on your machine. The author receives nothing.
